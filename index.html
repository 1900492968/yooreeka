Yooreeka

The Yooreeka project started with the code of the book "Algorithms of the Intelligent Web" (Manning 2009). Although the term "Web" prevailed in the title, in essence, the algorithms are valuable in any software application.

An Errata page for the book has been posted here.
The second major revision of the code will introduce some enhancements, some new features, and it will restructure the packages from the root org.yooreeka.*
You can find the Yooreeka 2.0 API (Javadoc) here
Lastly, Yooreeka 2.0 will be licensed under the Apache License rather than the somewhat more restrictive LGPL.

The library is written 100% in the Java language.
